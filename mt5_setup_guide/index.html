Step 2: Python Environment Setup

2 / 5

Install Required Python Libraries

The Python AI engine requires several libraries for machine learning and data processing.

1. Check Python Installation

First, verify that Python 3.7+ is installed on your system:

python --version

If Python is not installed, download it from python.org

2. Install Required Libraries

Run the following command to install all dependencies:

pip install pandas numpy scikit-learn joblib
pandas - Data manipulation and analysis
numpy - Numerical computing
scikit-learn - Machine learning algorithms
joblib - Model persistence

3. Verify Installation

Test if all libraries are installed correctly:

python -c "import pandas, numpy, sklearn, joblib; print('All libraries installed!')"

Step 3: MetaTrader 5 Configuration

3 / 5

Configure MT5 for AI Engine Communication

MetaTrader 5 needs to be configured to allow socket communication with the Python AI engine.

Step 1: Open MT5 Options

  1. Open MetaTrader 5
  2. Click on Tools in the menu bar
  3. Select Options

Step 2: Configure Expert Advisors

  1. In the Options window, go to the Expert Advisors tab
  2. Check the following boxes:
    • ✓ Allow automated trading
    • ✓ Allow DLL imports
    • ✓ Allow WebRequest for listed URL
  3. In the URL list, add: http://127.0.0.1:8888

📸 MT5 Options - Expert Advisors Tab

(Check "Allow DLL imports" and "Allow WebRequest")

Step 3: Compile the Expert Advisor

  1. Open the MetaEditor (Ctrl+Shift+E in MT5)
  2. Open the file Improved_SelfLearning_EA.mq5
  3. Click Compile (F7) or use File → Compile
  4. Ensure there are no errors in the compilation

Step 4: Running the System

4 / 5

Start the Python Engine and Attach the EA

Follow the steps below to launch the complete system. Important: Start the Python engine FIRST, then attach the EA in MT5.

Step 1: Start the Python AI Engine

Open a terminal/command prompt and navigate to your project directory, then run:

python self_learning_engine.py

You should see: Self-Learning Engine started on 127.0.0.1:8888

Step 2: Attach the EA to MT5

  1. In MetaTrader 5, open a chart (any symbol and timeframe)
  2. Right-click on the chart and select Expert Advisors → Manage
  3. Click Add and select Improved_SelfLearning_EA
  4. Click OK

Step 3: Monitor the Connection

Once the EA is attached, you should see in the Python console:

MT5 Connected: ('127.0.0.1', ...)
Successfully learned from X trades
✓ System Running!

The Python engine is now learning from your MT5 trades and improving the strategy in real-time.

Performance Monitoring

Check your strategy's performance by reviewing:

  • Win Rate: Percentage of profitable trades
  • Trade Count: Number of trades executed
  • AI Confidence Score: How confident the AI is in each trade (0-1)
  • Dynamic Threshold: Minimum score required to approve trades

Troubleshooting

❌ "Connection refused" error

Make sure the Python engine is running on port 8888. Check that no other application is using this port.

❌ "Not enough data to learn yet"

The system needs at least 50 trades to start learning. Keep trading until you accumulate enough history.

❌ EA not executing trades

Verify that "Allow automated trading" is enabled in MT5 Options and that the AI confidence score is above the threshold.

💡 Pro Tip:

The more trades you accumulate, the smarter the AI becomes. Consider running the system continuously for best results!
